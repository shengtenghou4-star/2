# 命镜 · 八字排盘、证据与力量裁决引擎

命镜按“确定性排盘 → 结构关系 → 月令根气 → 作用路径 → 力量候选”的顺序逐层工作。每一层都保存来源、规则 ID、反证与未决项；大模型不参与排盘，也无权篡改底层事实。

## 当前能力（v0.6）

### 历法与时间

- 公历输入
- 农历输入及闰月校验
- 农历先换算为对应民用公历，再进行时间修正
- 出生地点、经纬度、UTC 标准时差和夏令时显式输入
- 真太阳时修正：夏令时回退、经度修正、均时差分别记账
- 民用时盘与真太阳时盘并列生成、逐柱对照
- 立春定年、节气定月
- 晚子时 `23:00` 换日与 `0:00` 换日两种口径

### 原局四柱

每一柱均包含：

- 天干、地支、阴阳与五行
- 天干十神
- 地支藏干、藏干十神及本气／中气／余气顺序
- 纳音
- 十二长生
- 旬与旬空
- 基础神煞落点

### 大运、流年与流月

- 大运顺逆、起运时间、九步大运
- 每步大运的十神、藏干、纳音、十二长生、旬空和神煞
- 每一流年的完整柱信息，不只显示干支
- 每年十二流月
- 流月严格按照十二“节”划分：立春起寅月，惊蛰起卯月，依次到小寒起丑月
- 每个流月均保存准确起止交节时间

### 神煞基础表

当前只收录规则清楚、可标明查法的基础神煞：

- 天乙贵人、文昌贵人
- 禄神、羊刃
- 驿马、桃花、华盖、将星
- 劫煞、亡神
- 红鸾、天喜

神煞只作为结构标签，不独立判断吉凶，也不凌驾于五行、月令与格局之上。存在流派差异的项目会在数据中注明。

### 干支与五行作用关系

原局及所选大运、流年、流月可以共同进入关系引擎。当前识别：

- 天干五合、天干相冲
- 地支六合、六冲、六害、六破
- 相刑、自刑、三刑
- 三合、三会
- 半合、拱合
- 显干与各柱地支本气之间的五行同类、相生、相克方向

同一对子若同时符合多种关系，系统并列保留，不互相覆盖。

### 月令与根气证据层

- 月支所属季节阶段
- 月支本气、中气、余气及对应十神
- 五行旺、相、休、囚、死基础状态
- 日主在月支的十二长生状态
- 日主以及各显干的根气索引
- 同干根与同类根严格分开
- 本气、中气、余气来源层级分别保存
- 藏干透出与同类显干分别保存
- 根所在支被合、冲、刑、害、破触及的关系索引
- 比劫、印星、食伤、财星、官杀出现账本
- 原局证据与大运、流年、流月新增证据分开统计

这些数据先回答“证据在哪里”。根所在支参与合冲，只表示结构被触及，不等于根已经被拔除、冲散或合化。

### 作用路径与制化条件层

- 原局四柱的相邻、隔柱、远隔位置
- 大运、流年、流月之间的层级邻接
- 岁运与原局的跨层接触
- 天干五合的合绊候选、合化条件候选与争合／受冲冲突
- 地支六冲的原局对冲、岁运冲动与辰戌丑未冲库／冲开候选
- 根所在支被六冲触及的证据数量
- 五行相克之间的通关元素及显干／藏干材料
- 官杀生印、食伤制杀、印制食伤、食伤生财、财生官杀等十神链候选
- 同一干支同时参与合与冲、刑、害、破时的冲突索引

“材料齐备”“条件候选”不等于作用已经成立，也不会静默选择互相冲突的某一条关系。

### 力量裁决器 v1

当前第一次允许输出旺衰候选，但必须连同完整证据链一起展示：

- 原局底盘与当前岁运叠加分开计算
- 月令旺相休囚死作为季节总纲
- 同干根／同类根、本中余气和柱位分别折算
- 比劫、印星进入扶身侧；食伤、财星、官杀进入耗泄克身侧
- 原局年、月、日、时采用不同位置因子
- 大运、流年、流月只作为当前态修正，不改写出生原局
- 岁运干支与原局重复时记录“重复引动”
- 根所在支逢合冲刑害破时折减可用性，但不直接归零
- 天干被合、被冲时折减其直接作用
- 通关、制化、合绊和冲开只作为低置信度二阶修正
- 同时生成身旺、偏强、中和、偏弱、身弱、从强、从弱候选
- 每个候选保存支持证据、反证和阻断条件
- 从强、从弱采用严格阻断：只要反向透干、根气或反证仍明显，就不能轻易领先
- 输出领先候选、候选排序、置信度和未决冲突

系统中的“证据单位”和“候选拟合度”是同一张盘内部的相对比较工具，不是假装客观物理测量，也不是统计学概率。界面不会只扔出一个无法复核的“身强 73 分”。

## 时间校准公式

```text
真太阳时 = 民用钟表时间 − 夏令时 + 4 ×（当地经度 − 时区中央经线）+ 均时差
时区中央经线 = UTC 标准时差 × 15°
```

当前均时差采用分钟级工程近似。历史时区和历史夏令时不由现代时区反推，必须按出生当时制度输入。

## 当前严格边界

力量裁决器 v1 仍不会判断：

- 最终格局名称及格局成败
- 合化、三合、三会是否已经完成真实转化
- 冲、刑、害、破对应的具体人生事件
- 调候用神、扶抑用神、通关用神的最终排序
- 喜忌五行及职业、婚姻、财富等现实断语
- 神煞的吉凶权重

下一层需要继续完善格局候选、调候与扶抑分轨、用神竞争，以及命局结论与岁运事件之间的分层推理。

## 架构原则

历法和四柱计算由确定性程序完成。规则引擎只读取结构化排盘与证据对象；大模型不参与排盘，也无权修改底层事实。所有现实断语必须引用规则引擎已经确认的事实与候选，不得凭空补盘。
