# 命镜 · 八字基础排盘引擎

命镜当前阶段只做一件事：把排盘、历法、十神、藏干、大运、流年、流月、神煞与干支作用关系做成稳定、透明、可审计的底层事实。旺衰、格局、喜忌和用神暂不越级加入。

## 当前能力（v0.3）

### 历法与时间

- 公历输入
- 农历输入及闰月校验
- 农历先换算为对应民用公历，再进行时间修正
- 出生地点、经纬度、UTC 标准时差和夏令时显式输入
- 真太阳时修正：夏令时回退、经度修正、均时差分别记账
- 民用时盘与真太阳时盘并列生成、逐柱对照
- 立春定年、节气定月
- 晚子时 `23:00` 换日与 `0:00` 换日两种口径

### 原局四柱

每一柱均包含：

- 天干、地支、阴阳与五行
- 天干十神
- 地支藏干、藏干十神及本气／中气／余气顺序
- 纳音
- 十二长生
- 旬与旬空
- 基础神煞落点

### 大运、流年与流月

- 大运顺逆、起运时间、九步大运
- 每步大运的十神、藏干、纳音、十二长生、旬空和神煞
- 每一流年的完整柱信息，不只显示干支
- 每年十二流月
- 流月严格按照十二“节”划分：立春起寅月，惊蛰起卯月，依次到小寒起丑月
- 每个流月均保存准确起止交节时间

### 神煞基础表

当前只收录规则清楚、可标明查法的基础神煞：

- 天乙贵人、文昌贵人
- 禄神、羊刃
- 驿马、桃花、华盖、将星
- 劫煞、亡神
- 红鸾、天喜

神煞只作为结构标签，不独立判断吉凶，也不凌驾于五行、月令与格局之上。存在流派差异的项目会在数据中注明。

### 干支与五行作用关系

原局及所选大运、流年、流月可以共同进入关系引擎。当前识别：

- 天干五合、天干相冲
- 地支六合、六冲、六害、六破
- 相刑、自刑、三刑
- 三合、三会
- 半合、拱合
- 显干与各柱地支本气之间的五行同类、相生、相克方向

同一对子若同时符合多种关系，系统并列保留，不互相覆盖。

## 时间校准公式

```text
真太阳时 = 民用钟表时间 − 夏令时 + 4 ×（当地经度 − 时区中央经线）+ 均时差
时区中央经线 = UTC 标准时差 × 15°
```

当前均时差采用分钟级工程近似。历史时区和历史夏令时不由现代时区反推，必须按出生当时制度输入。

## 严格边界

当前基础层不会判断：

- 天干五合是否真正合化
- 三合、三会是否成局成势
- 冲、刑、害、破是否有力以及现实应事
- 五行生克是否因通根、月令、透藏、合绊或通关而发生改变
- 神煞的吉凶权重
- 身强身弱、格局、调候、喜忌和用神

这些内容必须等待月令、根气、透干、制化和冲突裁决规则完善后进入下一层。

## 架构原则

历法和四柱计算由确定性程序完成。后续规则引擎只读取结构化排盘结果；大模型不参与排盘，也无权修改底层事实。
