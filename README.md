# 命镜 · 八字排盘内核

命镜的第一阶段：一个不依赖大模型、结果可导出、边界可核对的八字排盘工作台。

## 当前能力

- 公历出生日期和时间排盘
- 立春定年、节气定月
- 年柱、月柱、日柱、时柱
- 天干十神、地支藏干及藏干十神
- 五行、纳音、十二长生、旬空
- 胎元、胎息、命宫、身宫
- 大运顺逆、起运时间、九步大运和逐年流年
- 晚子时 23:00 换日 / 0:00 换日两种口径
- 排盘结果复制为结构化 JSON

## 暂未加入

- 真太阳时修正
- 农历输入
- 刑冲合害关系图

## 本地运行

```bash
npm install
npm run dev
```

验证：

```bash
npm test
npm run build
```

## 架构原则

历法和四柱计算由确定性程序完成。未来的命理规则引擎只读取 `BaziChart` 结构化结果；大模型不参与排盘，也无权修改底层事实。
